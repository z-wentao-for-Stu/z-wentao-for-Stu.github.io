
# 学术主页（静态站点）
- 入口文件：`index.html`
- 使用方式：
  1. 打开此压缩包，解压得到文件夹；
  2. 直接双击 `index.html` 本地预览；
  3. 部署到 GitHub Pages：
     - 将 `index.html` 放在仓库根目录（或 `docs/` 目录）；
     - 仓库 Settings → Pages → 选择分支（main）和路径（/ 或 /docs），保存；
  4. 也可以部署到任意静态托管（如 Vercel、Netlify、阿里云 OSS、COS 等）。

## 语言切换
- 通过切换 `body` 的类名 `lang-en` 来控制 `.zh` / `.en` 的显示。
- 默认中文；点击右上按钮切换为英文。
